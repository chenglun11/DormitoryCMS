### Requirements
所有依赖基于Maven管理
请务必通过 [前端地址](https://github.com/chenglun11/Dormitoryfront) 打开


### Getting Started
1. Clone本项目并从IDEA打开
2. 使用IDEA自带的Maven管理器安装依赖
3. 将SQL导入数据库(其中测试数据自行处理)
4. 运行✅

### Version Control
